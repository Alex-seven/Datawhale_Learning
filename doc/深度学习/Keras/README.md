# Keras
**课程设计**：有容

**组队学习说明**：通过阅读和练习keras中文文档，学习keras的基本模块，侧重于对keras的各模块的了解，主要参考资料为keras中文文档。

**难度系数**：简单

**组队学习周期**：6天

**每个任务完成大概所需时间**：2-3h
